# MacOS screen resolution tool

## Basic Usage

```
$ cargo run -- -h
MacOS Screen Resolution Tool 0.1.0
Bernard Niset
Allows to list, get and set screen resolutions.

USAGE:
    screenresolution-rs [SUBCOMMAND]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

SUBCOMMANDS:
    get     Get current active resution for current display (TODO)
    help    Prints this message or the help of the given subcommand(s)
    list    List available resolutions for current display
    set     Set current active resution for current display (TODO)
```
